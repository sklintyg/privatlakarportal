Välkommen till Webcerts registrering för privatläkare

Såhär går registreringen till:

  • Du fyller i uppgifter om dig själv och din verksamhet. Uppgifterna behövs för att du ska kunna använda alla funktioner i Webcert.
  • Din yrkeslegitimation verifieras automatiskt mot Socialstylrelsens register över legitimerad hälso- och sjukvårdspersonal (HOSP).
  • Du godkänner Användarvillkoren och Inera AB, som förvaltar tjänsten, godkänner dig som användare. Verifieringen sker omedelbart om din yrkesklegitimation finns regiistrerad i Hälso- och sjukvårdens adressregister (HSA).
  • Du kan börja använda Webcert.
